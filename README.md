# Tetris

### To compile:

    javac tetris/game/*.java tetris/gui/*.java

or (JAR):

    javac tetris/game/*.java tetris/gui/*.java
    jar cfm Tetris.jar MANIFEST.MF tetris/game/*.class tetris/gui/*.class

### To run:

    java tetris/gui/TetrisFrame

or (JAR):

    java -jar Tetris.jar
